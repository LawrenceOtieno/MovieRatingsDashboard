### 1. Clone the repository

```bash
git clone https://github.com/Lajema/M-pesa_performace_dashboard.git
cd m-pesa-performance-dashboard
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run app.py
```
