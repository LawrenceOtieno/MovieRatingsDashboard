### 1. Clone the repository

```bash
https://github.com/LawrenceOtieno/MovieRatingsDashboard.git
cd MovieRatingsDashboard
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run app.py
```
