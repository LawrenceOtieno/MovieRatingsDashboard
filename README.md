This project solves data accessibility barriers by providing a centralized, visual interface for complex cinematic datasets. It is designed for data analysts and film researchers and aims to streamline the identification of trends in audience preferences and genre performance.
The Movie Ratings Dashboard is built with Python and Streamlit using a synthetic dataset of 100 movies.

## Features
- Interactive Streamlit dashboard
- Average ratings by genre and director
- Top-rated movies table
- Fully synthetic, realistic dataset
  
## Tech Stack
- Python
- Pandas
- Streamlit
- Matplotlib (optional)

## How to Run

# Clone the repo
git clone https://github.com/LawrenceOtieno/MovieRatingsDashboard.git
cd MovieRatingsDashboard

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py

## Deplyment
The dashboard is deployed on Streamlit Cloud. Ensure the following are included in your requirements.txt:

streamlit
pandas
plotly

## Key Insights Highlighted

# Highest-rated movies consistently come from drama and sci-fi genres

# Average ratings vary significantly across release decades

# Viewer engagement spikes during weekends and holidays

# Certain directors show a strong correlation with high movie ratings
